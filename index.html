Rôle des balises

  • <iframe>: Sert à intégrer une autre page web à l'intérieur de l'actuelle , exemple :


  • <fieldset> : Utilisée pour regrouper des éléments d'un formulaire , exemple :
    Informations personnelles

  • <address> : Fournit des informations de contact , exemple :
    Contactez-moi à : info@example.com.


  • <abbr>: Déclare une abréviation ou un acronyme , exemple :
    Hello, my name is Mohamed

  • <cite> : Indique le titre d'une œuvre , exemple :
    un type de text

  • <code> : Affiche du code informatique , exemple :
    console.log('Hello, world!')

  • <ins> : Marque un texte inséré , exemple :
    Ce texte a été ajouté.

  • <del> : Marque un texte supprimé , exemple :
    Ce texte a été supprimé.

  • <q> : Utilisée pour les courtes citations , exemple :
    Today is good.

  • <pre> : Affiche du texte préformaté , exemple : 
        Text with       spaces
    and line breaks
  • <output> : Représente le résultat d'un calcul ou d'une action , exemple :
    42
